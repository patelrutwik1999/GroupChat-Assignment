Hello,

This is a groupchat Application, which consits of two packages one is server and other is client.

To compile this file, one should enter:- javac *.java, to compile the file in cmd.
To run this file, one should enter:- java -cp . package_name/file_name portNumber, to run this file.

Note :- Here portNumber in server file would be the port number on which the chat is to be held
In client, it should be the specified port, where client need to join the chat.
